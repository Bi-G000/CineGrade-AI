# Application Icon Specification

This directory must contain the main application icon used by the CineGrade AI plugin. 
Binary image files cannot be generated by this script. Please create or export the 
icon according to the specifications below and place it in this `resources/icons/` directory.

## Official Filename
`app_icon_256x256.png`

## Format & Technical Requirements
*   **Format:** PNG (Raster)
*   **Dimensions:** Exactly 256x256 pixels
*   **Color Profile:** sRGB
*   **Background:** Strictly transparent (Alpha channel required) to ensure compatibility with both Light and Dark UI themes in Adobe Illustrator.

## Design Specification
The icon should visually communicate "Cinema-Grade Color Grading" and "Precision".

*   **Visual Motif:** A stylized, abstract representation of a Curves graph or a 3-Way Color Wheel (Shadows/Midtones/Highlights). 
*   **Color Palette:** 
    *   Deep Teal/Blue (`#2F80ED`) for Shadows
    *   Neutral/White (`#E0E0E0`) for Midtones
    *   Warm Orange (`#F2994A`) for Highlights
*   **Style:** Flat design, modern, minimal drop shadows. Must be legible at small sizes (e.g., 24x24 in the UXP panel header).
*   **Safe Zone:** Maintain a 16px padding (10% safe area) on all sides to prevent the graphic from touching the edges or being clipped when rendered in operating system docks or Adobe's panel UI.

## Instructions
1. Design or export the icon using software like Adobe Illustrator or Figma.
2. Ensure the artboard is exactly 256x256px.
3. Export as PNG with "Transparent Background" enabled.
4. Verify the filename is exactly `app_icon_256x256.png`.
5. Place the file in this `resources/icons/` directory.
